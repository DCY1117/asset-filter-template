# ML Browser UI Sections (Legacy)

- `docs/ui/ml-browser-app-legacy/sections/ml-assets-browser.md` - Main assets list, filters, and actions.
- `docs/ui/ml-browser-app-legacy/sections/asset-detail.md` - Legacy local asset detail route (`/assets/:id`).
- `docs/ui/ml-browser-app-legacy/sections/asset-create.md` - Asset creation flow.
- `docs/ui/ml-browser-app-legacy/sections/catalog-browser.md` - Catalog list view.
- `docs/ui/ml-browser-app-legacy/sections/catalog-detail.md` - Catalog detail and contract offers.
- `docs/ui/ml-browser-app-legacy/sections/contracts-list.md` - Contract definitions list.
- `docs/ui/ml-browser-app-legacy/sections/contract-definition-create.md` - Create contract definitions.
- `docs/ui/ml-browser-app-legacy/sections/model-execution.md` - Inference UI.
