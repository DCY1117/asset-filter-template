# UI Documentation

UI docs are split into two tracks:

- `docs/ui/ml-browser-app-legacy/`:
  Documentation for the previous custom UI in `asset-filter-template/ui/ml-browser-app`.
- `docs/ui/data-dashboard/`:
  Documentation for the current GUI based on `DataDashboard` (new default UI).

Use `docs/ui/data-dashboard/` as the source of truth for current behavior.
